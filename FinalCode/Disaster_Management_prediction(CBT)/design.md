# Natural Disaster Prediction System - Design Guide

## Design Philosophy

### Color Palette
- **Primary**: Deep Ocean Blue (#1e3a8a) - Represents reliability and trust
- **Secondary**: Emergency Orange (#ea580c) - For alerts and warnings
- **Accent**: Forest Green (#059669) - For safe conditions and nature
- **Neutral**: Warm Gray (#6b7280) - For text and subtle elements
- **Background**: Off-white (#fefefe) - Clean, professional backdrop

### Typography
- **Display Font**: "Playfair Display" - Elegant serif for headings and hero text
- **Body Font**: "Inter" - Clean, modern sans-serif for readability
- **Monospace**: "JetBrains Mono" - For code and technical data

### Visual Language
- **Minimalist Approach**: Clean, uncluttered interface focusing on functionality
- **Data-Driven Design**: Emphasis on clear data visualization and metrics
- **Professional Authority**: Scientific credibility with modern aesthetics
- **Accessibility First**: High contrast ratios and clear visual hierarchy

## Visual Effects & Styling

### Used Libraries
1. **Anime.js** - Smooth page transitions and element animations
2. **ECharts.js** - Interactive data visualization and charts
3. **p5.js** - Creative coding for background effects and simulations
4. **Splide.js** - Image carousels and content sliders
5. **Matter.js** - Physics simulations for particle effects
6. **PIXI.js** - Advanced visual effects and GPU acceleration
7. **Shader-park** - Custom shaders for background effects

### Animation & Effects
- **Hero Background**: Animated particle system representing seismic waves
- **Text Effects**: Gradient color cycling for key headings
- **Data Visualization**: Smooth chart animations with staggered reveals
- **Hover States**: Subtle 3D transforms and shadow effects
- **Loading States**: Skeleton screens with shimmer effects

### Header Effect
- **Navigation**: Fixed transparent header with backdrop blur
- **Hero Section**: Full-screen animated background with floating elements
- **Typography**: Large, bold headings with gradient text effects
- **Call-to-Action**: Prominent buttons with hover animations

### Interactive Elements
- **Cards**: Elevated surfaces with hover lift effects
- **Forms**: Floating labels with focus animations
- **Buttons**: Gradient backgrounds with ripple effects
- **Charts**: Interactive tooltips and smooth transitions
- **Maps**: Zoom and pan controls with smooth animations

### Background Design
- **Consistent Theme**: Subtle particle system throughout all pages
- **Depth Layers**: Multiple parallax layers for visual interest
- **Color Gradients**: Subtle gradients from primary to secondary colors
- **Texture**: Light noise texture for organic feel

### Mobile Responsiveness
- **Breakpoints**: Tailwind CSS responsive utilities
- **Touch Interactions**: Larger touch targets and gesture support
- **Performance**: Optimized animations for mobile devices
- **Navigation**: Collapsible menu with smooth slide animations

### Accessibility Features
- **High Contrast**: 4.5:1 minimum contrast ratio for all text
- **Focus States**: Clear keyboard navigation indicators
- **Screen Readers**: Semantic HTML and ARIA labels
- **Motion Preferences**: Respect user's reduced motion settings
